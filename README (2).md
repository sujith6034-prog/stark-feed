# ⚡ STARK Feed
### AI-Powered Niche Intelligence · A ChromeCraft App

> **Blunt. Raw. Real-time.**  
> STARK Feed cuts through the noise — fetching live news for your niche and turning it into a sharp 3-point analyst brief via Gemini AI.

---

## ✨ Features

- 📡 **Live News** — Pulls 5 fresh articles per niche via NewsData.io
- 🤖 **AI Analyst Briefs** — Gemini 1.5 Flash responds as a blunt senior analyst
- 🧠 **3-Part Format** — Every brief: `The Reality · The Impact · The Move`
- 💾 **Persistent Niches** — Your niche list saves across app restarts
- ➕ **Add Any Niche** — Track any topic in real time
- 📋 **Copy to Clipboard** — One-tap copy of the full brief
- 🌑 **ChromeCraft UI** — Deep dark theme with cyan/blue neon accents

---

## 🚀 Quick Start

```bash
git clone https://github.com/YOUR_USERNAME/stark-feed.git
cd stark-feed
pip install flet newsdataapi google-genai
flet run main.py
```
